# Employee Management System

A full-stack application for managing employee information, built with JavaScript, Java, HTML, and CSS.

## 📋 Overview

This Employee Management System provides a comprehensive solution for organizing and managing employee data. The application features a modern web interface combined with robust backend processing to ensure efficient employee record management.

## 🛠️ Technology Stack

- **Frontend:** JavaScript (60.5%), HTML (1.7%), CSS (0.9%)
- **Backend:** Java (36.9%)

## ✨ Features

- Employee record creation and management
- Data storage and retrieval
- User-friendly web interface
- Secure backend processing

## 🚀 Getting Started

### Prerequisites

- Java (for backend)
- Node.js/npm (for frontend)
- Git

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SharvariKarmalkar3244/Employee-Management-System.git
   cd Employee-Management-System
   ```

2. **Setup Backend (Java):**
   ```bash
   # Navigate to backend directory
   cd backend
   # Build the project (if using Maven)
   mvn clean install
   # Or if using Gradle
   gradle build
   ```

3. **Setup Frontend (JavaScript):**
   ```bash
   cd frontend
   npm install
   npm start
   ```

## 💻 Usage

1. Start the Java backend server
2. Launch the frontend application
3. Access the application in your web browser
