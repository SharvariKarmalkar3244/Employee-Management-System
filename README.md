# Employee Management System

A full-stack application for managing employee information, built with React, Spring Boot, HTML, and CSS.

## 📋 Overview

This Employee Management System provides a comprehensive solution for organizing and managing employee data. The application features a modern React-based web interface combined with a robust Spring Boot backend to streamline employee management operations.

## 🛠️ Technology Stack

- **Frontend:** React, Bootstrap, JavaScript (60.5%), HTML (1.7%), CSS (0.9%)
- **Backend:** Spring Boot, Java (36.9%)

## ✨ Features

- Employee record creation and management
- Data storage and retrieval
- User-friendly web interface with React and Bootstrap
- Secure backend processing with Spring Boot

## 🚀 Getting Started

### Prerequisites

- Java 8+ (for Spring Boot backend)
- Node.js/npm (for React frontend)
- Git

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SharvariKarmalkar3244/Employee-Management-System.git
   cd Employee-Management-System
   ```

2. **Setup Backend (Spring Boot):**
   ```bash
   # Navigate to backend directory
   cd backend
   # Build the project with Maven
   mvn clean install
   # Run the Spring Boot application
   mvn spring-boot:run
   ```

3. **Setup Frontend (React):**
   ```bash
   cd frontend
   npm install
   npm start
   ```

## 💻 Usage

1. Start the Spring Boot backend server (runs on http://localhost:8080 by default)
2. Launch the React frontend application (runs on http://localhost:3000 by default)
3. Access the application in your web browser
4. Begin managing employee records using the intuitive Bootstrap-styled interface

## 🔧 Configuration

Configure application settings:
- Backend server port (default: 8080)
- Database connection in application.properties
- Frontend API endpoints (proxy to backend in package.json)
